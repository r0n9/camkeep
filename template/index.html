配置管理

集中管理摄像头接入源、录像策略、动检与延时任务。保存后会写入配置并热重启录像任务。

表单优先 支持 YAML 高级编辑 保存后自动应用
配置说明

从 go2rtc 导入摄像头

扫描 go2rtc 中尚未加入 CamKeep 的流。

1 先准备流 在 go2rtc 中添加或确认已有摄像头流。
2 扫描可导入 只显示还没有加入 CamKeep 的流。
3 加入后保存 加入配置不会立即生效，需要保存并应用。
导入只会追加摄像头表单，检查参数无误后再保存。

每日按小时合并

在低峰时段将前一天录像按自然小时合并为 MP4，成功后删除对应切片。

摄像头配置

按 order 排序展示；常用参数直接填写，动检流可单独配置低码率子码流。

批量添加摄像头

每行一个摄像头，支持 id 接入源 或只填接入源自动生成 ID。

批量输入
格式 1：摄像头ID 接入源 格式 2：接入源，系统按 host 自动生成 ID
粘贴多行接入源后会在这里预览。
暂无摄像头配置，点击“添加摄像头”创建。
